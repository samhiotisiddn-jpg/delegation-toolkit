🚀 QUICK START OPTIONS
════════════════════════════════════════════════════════════════════════════════

OPTION A: FASTEST (Use what you have)
──────────────────────────────────────────────────────────────────────────────
You're already using Claude via claude.ai. Get an API key:

1. Go to: https://console.anthropic.com/
2. Create account (if needed)
3. Go to API Keys section
4. Create new key
5. Add to your .env file:
   
   nano ~/fractalmesh_native/config/.env
   
   # Add this line:
   ANTHROPIC_API_KEY="sk-ant-your-key-here"

COST: ~$0.003 per query (0.3 cents)
BENEFIT: Best quality, fast responses, same AI you're using now


OPTION B: 100% FREE (No API keys needed)
──────────────────────────────────────────────────────────────────────────────
Install Ollama to run AI locally on your device:

1. Run the setup script:
   
   bash setup_opensource_ai.sh
   
2. It will:
   - Install Ollama in proot-distro
   - Download a local AI model (2-8GB)
   - Start the server automatically
   - Configure your .env file

3. That's it! The system will use your local AI.

COST: $0 (completely free)
BENEFIT: Privacy, no internet required, no rate limits
TRADEOFF: Slower, uses device resources


OPTION C: BEST OF BOTH (Multiple providers)
──────────────────────────────────────────────────────────────────────────────
Use multiple providers for redundancy:

1. Set up Anthropic (Option A) for quality
2. Install Ollama (Option B) as free backup
3. Optionally add HuggingFace for variety

The system automatically uses the best available option.


🔑 HOW TO GET API KEYS
════════════════════════════════════════════════════════════════════════════════

1️⃣  ANTHROPIC CLAUDE (RECOMMENDED - You're already using it!)
──────────────────────────────────────────────────────────────────────────────
Website: https://console.anthropic.com/
Cost: $3 per 1M input tokens, $15 per 1M output tokens
       (~$0.003 per trading query)

SETUP:
  a) Go to https://console.anthropic.com/
  b) Sign in (same account as claude.ai)
  c) Click "API Keys" in sidebar
  d) Click "Create Key"
  e) Copy the key (starts with "sk-ant-")
  f) Add to .env: ANTHROPIC_API_KEY="sk-ant-..."

FREE CREDIT: $5 free credit for new accounts
RATE LIMIT: 50 requests/minute (more than enough)


2️⃣  OLLAMA (LOCAL - 100% FREE)
──────────────────────────────────────────────────────────────────────────────
Website: https://ollama.com/
Cost: $0 (runs on your device)

SETUP:
  a) Run: bash setup_opensource_ai.sh
  b) Choose model size (llama2 recommended)
  c) Wait for download (one time)
  d) Server starts automatically

NO API KEY NEEDED - Runs locally!

MODELS:
  - llama3.2:1b  → 1GB (fastest, basic)
  - llama2       → 4GB (balanced) ← RECOMMENDED
  - mistral      → 4GB (better quality)
  - llama3       → 8GB (best quality)


3️⃣  HUGGING FACE (FREE TIER)
──────────────────────────────────────────────────────────────────────────────
Website: https://huggingface.co/
Cost: FREE (with rate limits)

SETUP:
  a) Go to https://huggingface.co/join
  b) Create free account
  c) Go to https://huggingface.co/settings/tokens
  d) Click "New token"
  e) Name it "fractalmesh"
  f) Select "read" permission
  g) Copy the token (starts with "hf_")
  h) Add to .env: HUGGINGFACE_TOKEN="hf_..."

FREE TIER: 
  - 1000 requests/hour per model
  - Access to 1000+ open source models
  - No credit card required


4️⃣  OPENROUTER (MULTI-PROVIDER)
──────────────────────────────────────────────────────────────────────────────
Website: https://openrouter.ai/
Cost: Varies by model (some free options)

SETUP:
  a) Go to https://openrouter.ai/
  b) Sign in with Google/GitHub
  c) Go to https://openrouter.ai/keys
  d) Click "Create Key"
  e) Copy the key (starts with "sk-or-")
  f) Add to .env: OPENROUTER_API_KEY="sk-or-..."

FREE MODELS:
  - meta-llama/llama-3-8b-instruct:free
  - google/gemma-7b-it:free
  - mistralai/mistral-7b-instruct:free


5️⃣  OPENAI (LAST RESORT)
──────────────────────────────────────────────────────────────────────────────
Website: https://platform.openai.com/
Cost: $10 per 1M input tokens ($0.01 per query)

ONLY use if you already have an OpenAI account.
The system deprioritizes this - it's the last fallback.
