## Description (copy into Gumroad)

Most people think you need a VPS or cloud server to run autonomous agents.

You don't. This guide shows you how to run 13 independent Python agents
simultaneously on a $200 Android phone using Termux — within a 250MB memory budget.

**What you get:**

- Complete Termux setup guide (proot-debian, Python 3, PM2)
- Memory-optimised agent template (runs in under 20MB each)
- PM2 ecosystem config for managing 13+ processes
- SQLite database schema for multi-agent coordination
- Flask dashboard for real-time monitoring
- Bash scripts for deploy, repair, clean, and restart
- Working example: geosignal agent reading live ISS position + Kp index

**Who this is for:**

- Developers who want to experiment with agent systems without paying for cloud
- Anyone learning Python automation on a tight budget
- Makers who want persistent background processes on Android

**What you need:**

- Android phone (4GB RAM recommended, 2GB minimum)
- Termux (free on F-Droid)
- Basic Python knowledge

No cloud accounts required. No credit card for servers. Runs offline.
